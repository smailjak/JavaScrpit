저번 수업때 탭을 클릭하였을때
노란색바와 상세내용을 보여주는 기능을 구현해보라고 하셨을때
"반드시 어떠한 이벤트가 발생되어야 .js파일내에 코드가 실행되겠구나!"라고 
생각하여 다음과 같이 코드를 작성해보았는데 의도한바와 같이 기능이 작동하지않더라구요
document.getElementbyId(' ').addEventLitener( 'click', function(){
	for( let index = 0; index < number; index++ )
	{
		...
	}
};);

그래서 선생님께서 알려주신 코드를 작성하여 
다시 구현해보니 당연히 제대로 작동하더군요.
for( let index = 0; index < number; index++ )
	{
		arrayList[ index ].addEventListener( 'click', function()
		{
			...
		};);
	}

제가 처음 작성한 코드와, 선생님께서 작성하신 코드의 차이가 무엇이며 
더불어 .js파일 내에 javascript 코드가 어떠한 방식으로 동작하게 되는지 궁금합니다.




- 브라우저가 HTML을 위에서 한줄한줄 읽다가 <script src="tab.js">를 만나면 tab.js 파일을 서버에 요청해서 다운받아와서 한줄한줄 바로 실행시킵니다. 자바스크립트에 뭐 항상 이벤트 조건을 줘야 실행시키는건 아니고 <script>안에 코드를 짜든 src로 넣든 항상 그 자리에서 실행시킵니다. 

- 전체코드를 보여주시면 아마 설명이 더 빠를거같은데여 음 첫째코드는 어떤 요소에 클릭이 일어났을 경우~ for반복문 내부의 코드를 X회 실행시켜주세요~ 라는 것이고 둘째 코드는 for 반복문을 이용해 내부 코드를 X회 실행시켜주라는 것 밖에 없습니다. 

- 둘째 코드는 탭기능 같은데 그냥 버튼에 addEventListener를 붙이는 행위를 3번 반복해라~ 라는 뜻이라 각각 버튼마다 addEventListener가 부착되고 버튼기능이 잘 개발된거지 뭐 특별한건 없습니다.
